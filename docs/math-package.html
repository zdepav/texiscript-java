math package

Package with basic mathematical operations.

Filters

abs
Meaning: Computes absolute value of input.
Syntax: math.abs()
Argument types: x: any generator
Usage example: 
math.abs(curve.linear(0, 1, -1, 1))
normalize
Meaning: Clamps input between 0 and 1.
Syntax: math.normalize()
Argument types: x: any generator
Usage example: 
math.normalize(math.add($a, $b))
add
Meaning: Adds two or more inputs.
Syntax: math.add(inputs...)
Argument types: inputs: two or more generators
Usage example: 
math.add($a, $b, $c)
sub
Meaning: Subtracts one input from another.
Syntax: math.sub()
Argument types: a: any generator
b: any generator
Usage example: 
math.sub($a, 0.5)
mul
Meaning: Multiplies two or more inputs.
Syntax: math.mul(inputs...)
Argument types: inputs: two or more generators
Usage example: 
math.mul($a, $b, $c)
div
Meaning: Divides one input by another.
Syntax: math.div()
Argument types: a: any generator
b: any generator
Usage example: 
math.div($a, 2)
sin
Meaning: Computes sine.
Syntax: math.sin()
Argument types: x: any generator
Usage example: 
math.sin($input)
cos
Meaning: Computes cosine.
Syntax: math.cos()
Argument types: x: any generator
Usage example: 
math.cos($input)
random
Meaning: Returns a pseudo-random value. When used twice with the same seed and the same argument value it generates the same value.
Syntax: math.random()
Argument types: x: [optional] any generator, default value is 0
Usage example: 
math.random([x])
invert
Meaning: Computes 1 - input.
Syntax: math.invert()
Argument types: x: any generator
Usage example: 
math.invert($input)
