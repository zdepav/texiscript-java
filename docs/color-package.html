color package

Package with filters for basic color manipulation.

Filters

r
Meaning: Extracts the red component of the color generated by the base generator.
Syntax: color.r()
Argument types: base: any generator
Usage example: 
color.r($input)

rgb
Meaning: Combines red, green and blue components into a color.
Syntax: color.rgb()
Argument types: red: any generator
green: any generator
blue: any generator
Usage example: 
color.rgb(
    curve.linear(),
    perlin.noise(),
    transform.rrot(curve.linear())
)
rgba
Meaning: Combines red, green, blue and alpha components into a color.
Syntax: color.rgba()
Argument types: red: any generator
green: any generator
blue: any generator
alpha: any generator
Usage example: 
color.rgba(
    perlin.noise:1(),
    perlin.noise:2(),
    perlin.noise:3(),
    shape.circle(1, 0, 0.5, 0.5, 0.5)
)
hsl
Meaning: Combines hue, saturation and lightness into a color.
Syntax: color.hsl()
Argument types: hue: any generator
saturation: any generator
lightness: any generator
Usage example: 
color.hsl(curve.linear(), 1, perlin.noise())
hsla
Meaning: Combines hue, saturation, lightness and alpha into a color.
Syntax: color.hsla()
Argument types: hue: any generator
saturation: any generator

)

Syntax: color.setalpha()
Argument types: base: any generator
alpha: any generator
Usage example: 
color.setalpha($input, 1)
