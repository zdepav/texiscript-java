filter package

Package with various processing filters.

Filters

blur
Meaning: Blurs the generated image. This filter can be very slow.
If only horizontal strength is specified it will be applied to both directions.
Syntax: filter.blur()
Argument types: base: any generator
horizontal: [optional] any generator, expects values between 0 and 4
vertical: [optional] any generator, expects values between 0 and 4
tiled: [optional] boolean (0 or 1)
Usage example: 
filter.blur($input, 0.6, 0.3, $FALSE)
fastblur
Meaning: Faster blur implementation with less customization. Can decrease quality when combined with transformation.
Syntax: filter.fastblur()
Argument types: base: any generator
strength: [optional] any generator
Usage example: 
filter.fastblur($input, 0.5)
emboss
Meaning: Adds embossed effect onto an image. Optionally uses another generator as a heightmap.
Syntax: filter.emboss()
Argument types: base: any generator
heightmap: [optional] any generator
strength: [optional] any generator
Usage example: 
filter.emboss($input)
