perlin package

Package with filters based on perlin noise. If only one color is suplied to any filter, it will be used as the second color.

Filters

noise
Meaning: Generates basic perlin noise.
Syntax: perlin.noise()
Argument types: scale: [optional] number, default value is 1
color1: [optional] any generator, default value is 0
color2: [optional] any generator, default value is 1
curve: [optional] any generator, default value is curve.linear()
Usage example: 
perlin.noise([scale], [color1], [color2], [curve])
