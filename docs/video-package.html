video package

Package with filters that can be used with the Video command.

Filters

time
Meaning: Returns the curent video frame index normalized to range [0,1).
For example in video with 5 frames this filter would return 0, 0.2, 0.4, 0.6 and 0.8.
Syntax: video.time()
Usage example: 
video.time()
frame
Meaning: Returns the curent video frame length. The video.time filter will always return multiple of frame length.
For example in video with 5 frames this filter would return 0.2.
Syntax: video.frame()
Usage example: 
video.frame()
transition
Meaning: Defines a transition between two values starting at the first video frame and ending at the last one. A curve can be supplied to alter the transition speed.
Syntax: video.transition()
Argument types: startingValue: any generator
endValue: any generator
curve: [optional] any generator, default value is curve.linear()
Usage example: 
video.transition(0, 1, curve.sin(0, 0.5, 0, curve.sin(1, 0.5, 0, 1)))
