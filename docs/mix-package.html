mix package

Package with filters used to combine multiple images.

Filters

blend
Meaning: Blends two inputs using alpha-blending.
Syntax: mix.blend()
Argument types: a: any generator
b: any generator
Usage example: 
mix.blend($image1, $overlay)
quattro
Meaning: Fills a 2 by 2 grid with inputs.
Syntax: mix.quattro()
Argument types: topLeft: any generator
topRight: any generator
bottomLeft: any generator
bottomRight: any generator
Usage example: 
mix.quattro(#F00, #0F0, #0F0, #00F)
sides
Meaning: Splits output into 4 triangles filling each one using a different input.
Syntax: mix.sides()
Argument types: top: any generator
right: any generator
bottom: any generator
left: any generator
Usage example: 
mix.sides(0, 1, 0, 1)
choice
Meaning: Chooses between inputs based on the control input.
Syntax: mix.choice(control, inputs...)
Argument types: control: any generator
inputs: two or more generators
Usage example: 
mix.choice(perlin.noise(), #000, #800, #F80, #FF8, #FFF)
switch
Meaning: Interpolates between inputs based on the control input.
Syntax: mix.switch(control, inputs...)
Argument types: control: any generator
inputs: two or more generators
Usage example: 
mix.switch(perlin.noise(), #000, #800, #F80, #FF8, #FFF)
