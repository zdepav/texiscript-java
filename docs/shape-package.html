shape package

Package with basic shapes. First argument of most filter is the shape's color, second argument determines color outside of the shape. Bevel allows for a smooth color interpolation on the shape's edge, curve determines interpolation type.

Filters

rect
Meaning: Generates a rectangle or a square.
Syntax: shape.rect()
Argument types: color: any generator
background: any generator
left: any generator
top: any generator
width: any generator
height: any generator
bevel: [optional] any generator, default value is 0
curve: [optional] any generator, default value is curve.linear()
Usage example: 
shape.rect(#F00, #000, 0.2, 0.3, 0.6, 0.4)

Meaning: Generates a star. Stuffingis a fraction of the radius that will be used as the internal radius.
Syntax: shape.star()
Argument types: color: any generator
background: any generator
