curve package

Package with filters that generate various curves.
Curves can be used without arguments or with 2 or 4 arguments. First two arguments determine the start and end of the curve on the X axis (by default 0 and 1), the third the fourth argument determine minimal and maximal value that will be generated (by default 0 and 1).
