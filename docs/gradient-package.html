gradient package

Package with filters generating various gradients. A gradient consists of two or more stops and interpolates between them.

Filters

stop
Meaning: Used to specify colors in a gradient.
Syntax: gradient.stop()
Argument types: position: number
color: any generator
Usage example: 
gradient.stop(0.5, #FF0000)
linear
Meaning: Creates a linear gradient between two points.
Syntax: gradient.linear(x1, y1, x2, y2, stops...)
Argument types: x1: any generator
y1: any generator
x2: any generator
y2: any generator
stops: two or more generators
Usage example: 
gradient.linear(
    0, 0, 1, 0,
    gradient.stop(0.000, #C04040),
    gradient.stop(0.333, #40C040)
    gradient.stop(0.667, #4040C0)
    gradient.stop(1.000, #C04040)
)
radial
Meaning: Creates a radial gradient between concentric circles.
Syntax: gradient.radial(centerX, centerY, radius1, radius2, stops...)
Argument types: centerX: any generator
centerY: any generator
radius1: any generator
radius2: any generator
stops: two or more generators
Usage example: 
gradient.radial(
    0.5, 0.5, 0, 0.5,
    gradient.stop(0.000, #C04040),
    gradient.stop(0.333, #40C040)
    gradient.stop(0.667, #4040C0)
    gradient.stop(1.000, #C04040)
)
angular
Meaning: Creates a gradient around a point.
Syntax: gradient.angular(centerX, centerY, stops...)
Argument types: centerX: any generator
centerY: any generator
stops: two or more generators
Usage example: 
gradient.angular(
    0.5, 0.5,
    gradient.stop(0.000, #C04040),
    gradient.stop(0.333, #40C040)
    gradient.stop(0.667, #4040C0)
    gradient.stop(1.000, #C04040)
)
