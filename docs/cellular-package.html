cellular package

Package with filters that generate cellular textures. Each filter takes a metric argument, that expects one of the constants in this package as its value.

Filters

cells
Meaning: Generates a cellular texture using the formula a2.
Syntax: cellular.cells()
Argument types: density: [optional] integer between 2 and 256, default value is 20
metric: [optional] 0, 1, 2 or 3, default value is $cellular.M_EUCLIDEAN
color1: [optional] any generator, default value is #000000
color2: [optional] any generator, default value is #FFFFFF
chaos: [optional] number between 0 and 1, default value is 1
curve: [optional] any generator, default value is curve.linear()
Usage example: 
cellular.cells(0.075, $cellular.M_CHEBYSHEV)
sparks
Meaning: Generates a cellular texture using the formula a * b.
Syntax: cellular.sparks()
Argument types: density: [optional] integer between 2 and 256, default value is 20
metric: [optional] 0, 1, 2 or 3, default value is $cellular.M_EUCLIDEAN
color1: [optional] any generator, default value is #000000
color2: [optional] any generator, default value is #FFFFFF
chaos: [optional] number between 0 and 1, default value is 1
curve: [optional] any generator, default value is curve.linear()
Usage example: 
cellular.sparks(0.075, $cellular.M_MANHATTAN, 0, 1, 1, curve.invarc())
