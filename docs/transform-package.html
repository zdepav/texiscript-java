transform package

Package with transforming filters.

Filters

fisheye
Meaning: Creates a fish eye effect. Negative values of scale create inverted effect.
Syntax: transform.fisheye()
Argument types: source: any generator
centerX: any generator
centerY: any generator
radius: any generator
scale: any generator
Usage example: 
transform.fisheye($image, 0.5, 0.5, 0.5, 0.5)
depolar
Meaning: Transforms input from polar coordinates.
Syntax: transform.depolar()
Argument types: source: any generator
centerX: any generator
centerY: any generator
Usage example: 
transform.depolar($image, 0.5, 0.5)
polar
Meaning: Transforms input to polar coordinates.
Syntax: transform.polar()
Argument types: source: any generator
centerX: any generator
centerY: any generator
Usage example: 
transform.polar($image, 0.5, 0.5)
translate
Meaning: Offsets the input by the given distance.
Syntax: transform.translate()
Argument types: source: any generator
x: number
y: number
Usage example: 
transform.translate($image, 0.5, 0.5)
offset
Meaning: Offsets the input by the given distance minus 0.5 (so that supplying 0.5, 0.5 as x, y means no movement).
Syntax: transform.offset()
Argument types: source: any generator
x: any generator
y: any generator
Usage example: 
transform.offset($image, 0.7, 0.5)
rotate
Meaning: Scales the input around the given origin by the given angle. Angle values between 0 and 1 correspond to 0 to 360 degrees.
Syntax: transform.rotate()
Argument types: source: any generator
originX: any generator
originY: any generator
angle: any generator
Usage example: 
transform.rotate($image, 0.5, 0.5, 0.125)
rrot
Meaning: Rotates source by 90 degrees clockwise.
Syntax: transform.rrot()
Argument types: source: any generator
Usage example: 
transform.rrot($image)
hrot
Meaning: Rotates source by 180 degrees.
Syntax: transform.hrot()
Argument types: source: any generator
Usage example: 
transform.hrot($image)
lrot
Meaning: Rotates source by 90 degrees anti-clockwise.
Syntax: transform.lrot()
Argument types: source: any generator
Usage example: 
transform.lrot($image)
flipx
Meaning: Flips the input horizontally.
Syntax: transform.flipx()
Argument types: source: any generator
Usage example: 
transform.flipx($image)
flipy
Meaning: Flips the input vertically.
Syntax: transform.flipy()
Argument types: source: any generator
Usage example: 
transform.flipy($image)
scale
Meaning: Scales the input from the given origin. If only horizontal scale count is specified, it will be used for both directions.
Syntax: transform.scale()
Argument types: source: any generator
originX: any generator
originY: any generator
horizontalScale: any generator
verticalScale: [optional] any generator
Usage example: 
transform.scale($image, 0.5, 0.5, 2)
tiles
Meaning: Repeats the input specified number of times. If only horizontal repetition count is specified, it will be used for both directions.
Syntax: transform.tiles()
Argument types: source: any generator
horizontalRepetitions: number
verticalRepetitions: [optional] number
Usage example: 
transform.tiles($image, 4)
